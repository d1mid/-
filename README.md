# Бот по продаже сантехники

Проект представляет собой чат-бота для подбора сантехники. Бот помогает выбрать товар по категории, бюджету и назначению, подсказывает цену и характеристики моделей и может работать в локальном консольном режиме или через Telegram.

## Запуск

Установите зависимости:

```bash
pip install -r requirements.txt
```

Запуск в локальном консольном режиме:

```bash
python -m src.bot.main
```

Запуск в Telegram:

```bash
set TELEGRAM_BOT_TOKEN=ваш_токен
python -m src.bot.main --telegram
```
